## Log Care Schedule Unit Test Plan

## Overview
This test plan outlines the unit testing strategy for the log feature of the PlantProgress component, which displays plant watering and sunlight history using chart components. The component receives wateringHistory and sunlightLog via props and visualizes the data.

## Test Categories

- Confirm that the watering history Line chart renders correctly using the mock data provided.

- Ensure that chart output includes expected dates and watering amounts 

- Confirm that the sunlight history Bar chart renders correctly using the mock data provided.

- Ensure that chart output includes expected date and sunlight hours
