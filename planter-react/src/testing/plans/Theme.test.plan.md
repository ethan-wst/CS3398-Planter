# Theme Unit Test Plan

## Overview
This test plan outlines the unit testing strategy for the function createCustomTheme.

## Test Categories
- Test if function creates a light theme (with the correct colors) when useLightTheme is true.
- Test if function creates a dark theme (with the correct colors) when useLightTheme is false.
- Test if the function returns a Theme object regardless if useLightTheme is true or false.
