# SearchPlantDetails Unit Test Plan

## Overview
This test plan outlines the unit testing strategy for the function SearchPlantDetails.

## Test Categories
- Test if function renders HTML elements with correct plant details given valid information.
- Test if function renders correctly while plant details are still loading from API.
- Test if function renders correctly if API throws an error.
- Test if function renders correctly if no information is returned from the API.
- Test if function can correctly render if the API returns a single value for sunlight (as opposed to multiple).
